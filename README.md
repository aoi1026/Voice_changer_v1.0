# George Michael Voice Converter v1.0

**Convert any vocal audio into a George Michael–style voice.**

This application runs on your computer. You can convert one file at a time (Single) or many files in one go (Batch). No internet is required after installation.

---

## First time only: install the program

1. **Unzip** the project folder to a location you prefer (e.g. `Desktop` or `Documents`).
2. **Double‑click** `run-install.bat`.
3. Wait until the installer finishes (it may take several minutes). When it says installation is complete, you can close the window.

You only need to do this once. Do **not** run the program as Administrator.

---

## How to start the program

You can start it in two ways:

| What you want | What to double‑click |
|---------------|----------------------|
| **Window like a normal program** (recommended) | `run-desktop.bat` |
| **Open in your web browser** | `run-michale_voice_changer.bat` |

- If you see a message like “Please run run-install.bat first”, run `run-install.bat` once, then try again.
- The first time you run the desktop version, a window may take a few seconds to appear.

---

## Quick start

1. Start the program (see above).
2. Go to the **Single** tab to convert one file, or the **Batch** tab to convert several.
3. **Single:** Use “Upload Audio” to choose a file, then click **Convert**. The converted file will appear in “Export Audio” and be saved to the path you set.
4. **Batch:** Add your files with “Input Files” (or put files in the `assets\audios` folder), set the “Output Folder” where you want results, then click **Convert**.

Output is saved in the format you choose (default is **MP3**). For full details, options, and troubleshooting, see **MANUAL.md** in this folder.
